03_자바스크립트 es6문법_화살표 함수(function)

함수란? 일련의 내용을 작성하여 사용자가 필요할 때 호출하여 내용을 실행하는 실행문이다.

함수는 매개변수가 있는 함수와 매개변수가 없는 일반 함수가 있다.

1. 기존 함수 작성과 실행

    //1. 매개변수가 있는 함수
    function 함수명(매개변수1, 매개변수2......){
      실행할 내용;
    }
    //2. 매개변수가 없는 함수
    function 함수명(){
      실행할 내용;
    }
    //3.
    const 변수명 = function(){
      실행할 내용;
    }
    //함수 실행은 함수명을 호출하여 실행한다
    함수명();

2. es6문법에서 새롭게 추가된 화살표 함수 작성과 실행방법

    const 변수명 = () => {
      실행할 내용;
    }

화살표 함수를 사용하면 구문이 더 짧고 간결하게 작성이 가능하다.

3. 매개 변수가 한개인 유형

    //괄호가 생략가능
    함수명 = 매개변수 => {표현식};
    함수명 = (매개변수) => {표현식};

매개변수가 한개일 경우는 굳이 괄호에 담지 않아도 상관없다.

화살표 함수와 this키워드

  • this(이것) - 해당 키워드가 속한 객체를 참고하고자 할 때
  • 혼자 쓰여 있다면 전역 객체를 의미
  • 함수에서는 전역 객체를 의미
  • 매서드 내부에서는 매서드를 소유한 객체를 의미
  • 이벤트는 이벤트를 받는 객체 의미
